## Setting up the development envrionment

* Run the following commands in the terminal

```
1. npm install
2. cp .env.dist .env
3. cp. twitter_config.js.dist twitter_config.js
```

* Edit the twitter_config.js and the .env files to your variables

* Run `npm start`.
